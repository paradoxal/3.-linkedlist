Download and familiarize yourself with this materials package.

The materials package contains files related to other exercises in this week's exercise set. Ignore the other files for now while focusing on the file linkedlist.py.

Using that file as a starting point, implement a doubly linked list. Most of the structure is already defined, but there are a few unimplemented methods that require your attention.

Use dedicated guardian nodes that constitute the very front and very end of the list. You might name them for example h or head and z or tail. The list should always have these two nodes, and it should be considered empty, when these two are the only nodes in it. They should not contain any data inserted by the user.

Do not use any of Python's standard data structures (list, dict, ...). Using them in a solution will be considered cheating.
