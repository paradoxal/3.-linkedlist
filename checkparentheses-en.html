Before attempting this exercise, you must solve the earlier programming exercises of this problem set where you implemented a linked list and a stack as you will have to use these structures in this solution. You will also need the file checkparentheses.py included in the materials package for the linked list exercise.

Into the file checkparentheses.py implement a function that returns a boolean value (True or False) indicating whether parentheses in a string given to the function are correctly ordered.

There are a few input examples along with corresponding correct output in the file. Those should clear out confusion on what exactly is the task.

Hint: The entire solution file need not be longer than 30 lines.
