Before attempting this exercise, you must solve the linked list exercise. You will need the linked list implementation and the files from the materials package for that exercise in order to complete this one.

Implement the unimplemented methods in the class Stack in file stack.py using your linked list implementation created in the earlier exercise.

Note: While you are to test your implementation with your own linked list, implementation, the grader uses the model linked list. So do not have your stack rely on custom behaviour on the part of the linked list.
